# 🚀 GenAI Prompt Engineering using LangChain

## 📌 Project Overview
This project is part of the **Data Science Internship – February 2026**.

The objective is to build a **Mini Prompt Engine** using LangChain that converts user inputs into structured and reusable prompts instead of hardcoded strings.

---

## 🎯 Objectives
- Replace hardcoded prompts with dynamic PromptTemplates
- Build reusable and scalable prompt systems
- Handle multiple inputs (topic, audience, tone)
- Implement prompt variations (teaching, interview, storytelling)
- Design a structured prompt pipeline
- Add input validation for robustness

---

## 🛠️ Technologies Used
- Python 🐍
- LangChain (`langchain_core`)
- Jupyter Notebook / Google Colab

---

## 📂 Tasks Implemented

### ✅ Task 1: PromptTemplate
Converted a hardcoded function into a reusable PromptTemplate.

### ✅ Task 2: Multi-Input Prompt System
Handled multiple inputs such as topic, audience, and tone.

### ✅ Task 3: Prompt Variations Engine
Created different prompt styles:
- Teaching
- Interview
- Storytelling

### ✅ Task 4: ChatPromptTemplate
Implemented role-based chat prompts:
- Teacher
- Interviewer
- Motivator

### ✅ Task 5: Input Validation Layer
Validated inputs and assigned default values for invalid entries.

### ✅ Task 6: Prompt Generator
Built a dynamic function to generate prompts based on style.

### ✅ Task 7: Template Reusability Test
Demonstrated how one template can handle multiple inputs efficiently.

---

## 🔄 Pipeline Flow
User Input → Validation → Prompt Template → Generated Prompt Output

---

## 💡 Key Learnings
- Importance of reusable prompt design
- How to structure dynamic prompts using LangChain
- Building modular GenAI systems
- Handling user input validation
- Designing real-world prompt pipelines
